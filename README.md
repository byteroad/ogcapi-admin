# README

This site creates the admin documentation for the OGC API SDI, at DGT. Docs are built using [mkdocs](https://www.mkdocs.org/).

Build using:

`mkdocs serve `

Deploy using

`mkdocs gh-deploy`

The site is live at: https://byteroad.github.io/ogcapi-admin/
